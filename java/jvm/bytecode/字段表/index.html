0x 00 02：字段的访问控制符为私有

0x 00 05：名称的索引，索引为常量池的第5项，这里的字段名称为 a

0x 00 06 ：字段的描述，索引为常量池的第6项，根据定义字段的描述为其类型，I

0x 00 00 ：说明属性总数为0，故没有后面的属性项的信息。
